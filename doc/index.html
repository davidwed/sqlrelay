SQL Relay

SQL Relay is powerful database-access middleware consisting of server components that maintain persistent database connections and client API's for using them.

Server Component Features
Persistent database connection pooling - speeds up and increases the efficiency of database-driven web-based applications
Proxying - provides database access from unsupported platforms
Throttling - prevents databases from becoming overloaded
Load Balancing - distributes access over replicated or clustered databases
Query Routing and Filtering - conditionally sends queries to one database or another
Database Support - a wide variety commercial and open-source databases are supported

Client API Features
Substitution and bind variables
Multi-row fetches
Client-side result set caching
Suspended transactions
Language Support - a wide variety of programming languages are supported
Abstraction Layer Support - many database abstraction layers are supported
