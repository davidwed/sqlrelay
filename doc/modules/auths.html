Introduction

Authentication modules allow the SQL Relay server programs to log various bits of information as they run. A section of the sqlrelay.conf file indicates which modules to load and what parameters to use when logging.

The module attribute specifies which module to load.

Multiple authentication modules may be loaded by specifying multiple authentication tags.

At startup, the SQL Relay server creates instances of the specified authentication modules and initializes them. When a client connects, the server calls each module's "run" method, passing it the credentials supplied by the client...
