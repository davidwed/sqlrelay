Introduction

Query modules allow the SQL Relay server programs to intercept certain queries, process them, and return a result set, without involving the database.

The module attribute specifies which module to load.

Multiple query modules may be loaded by specifying multiple query tags.

At startup, the SQL Relay server creates instances of the specified query modules and initializes them. When a client connects, the server calls each module's "run" method, passing it the query supplied by the client...
